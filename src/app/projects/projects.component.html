Recursive Backtracking Maze

GUI-based (turtle interface). This algorithm creates a new maze from a grid of cells. A random starting cell is chosen and marked as visited. Then it repeats the following steps: If there are unvisited neighbors, choose a random one and remove the wall between them. Mark the new cell as visited, and if all neighbors have been marked as visited, back up until finding a cell that has unvisited neighbors.
